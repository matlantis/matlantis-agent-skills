# Validation Dataset : COD
[PFP Validation](/api/resource/documents/pfp-validation/versions/latest/index.html)で用いられている、Crystallography Open Database (COD) の結晶構造をもとにしたデータセットの一部を提供し、その扱い方を示します。
ご自身で追加の検証を行いたい時などにご利用ください。


## 構造最適化データセット
構造最適化後のデータセットは以下のようにして読み込むことが出来ます。


```python
from collections import defaultdict
import io
import zipfile

import ase.io
```


```python
with zipfile.ZipFile("./assets/validation_dataset_cod/cod_opt.xyz.zip", "r") as zf:    
    with io.BytesIO() as f:
        f.write(zf.read("cod_opt.xyz"))
        f.seek(0)
        atoms_opt_dict = {
            str(atoms.info["cod_id"]): atoms
            for atoms in ase.io.read(io.TextIOWrapper(f), index=":", format="extxyz")
        }
```


`atoms_opt_dict`は、COD IDを`key`とする`dict`です。検証に利用しているデータセットからランダムにピックアップした、1000の構造が含まれます。

## 微小変位データセット
微小変位データセットも同様に読み込むことが出来ます。読み込みに10秒程度かかります。


```python
atoms_spd_dict = defaultdict(lambda : [])
with zipfile.ZipFile("./assets/validation_dataset_cod/cod_spd.xyz.zip", "r") as zf:
    with io.BytesIO() as f:
        f.write(zf.read("cod_spd.xyz"))
        f.seek(0)
        for atoms_ in ase.io.read(io.TextIOWrapper(f), index=":", format="extxyz"):
            atoms_spd_dict[str(atoms_.info["cod_id"])].append(atoms_)
```


構造最適化データセットと同一のCOD IDから生成された構造です。一つのCOD IDに対して、複数の異なる微小変位を加えた構造があるため、`dict`の中身は`list`となっています。9202構造が含まれています。


```python
len(atoms_spd_dict), sum(len(v) for v in atoms_spd_dict.values())
```




    (1000, 9202)




COD IDが1008058である構造の1つを得ます。`ase.Atoms` objectが得られます。


```python
atoms_spd = atoms_spd_dict["1008058"][0]
atoms_spd
```
