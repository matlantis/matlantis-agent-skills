# Known Issues about PFP
このページでは、PFPに関する既知の問題と、その対応状況を記載します。 ここに掲載する情報は完全な一覧ではありませんが、ユーザーの計算結果や運用判断に影響し得る問題を優先して追記します。 各項目の説明を確認し、必要に応じて回避策や修正済みバージョンをご確認ください。


| Status | 修正したバージョン | 問題を確認しているバージョン | 説明 |
|---|---|---|---|
| Open | 未修正 | v7.0.0 / v8.0.0 | **一部の化合物で表面エネルギーの値が負になる**<br>v7.0.0のPBEモードにおけるNiO、およびv8.0.0のr2SCANモードにおけるCrについて、一部のミラー指数面で表面エネルギーが負になる問題を確認しています。原因としては、構造最適化における力の収束閾値を厳しく設定した場合に、表面再構成や相転移が起こり、構造が過度に安定化することで表面エネルギーが負になる可能性があると考えています。本問題はPFPの精度に起因するものと考えられ、v9.0.0での解決に向けて改良を検討しています。 |
| Open | 未修正 | v7.0.0 / v8.0.0 | **O2が室温で凝集する**<br>v7.0.0以降のPBEモード、およびv8.0.0のr2SCANモードにおいて、O2ガスのMDシミュレーションを実施した際、設定温度が400 K以下であるにもかかわらず、O2分子が凝集してしまう問題を確認しています。 原因としては、v7.0.0以降のPFPにおいて、酸素分子間の相互作用に強い引力が生じている可能性があると考えています。 本問題の解決に向けて、PFPの改良を検討しています。また、暫定的な対処法として、酸素ガスの計算を実施する場合は、v6.0.0のPBEモードの使用を推奨いたします。 |


# 記載方針
  * `Open`: 現在対応中、または修正バージョンがまだリリースされていない問題です。
  * `Closed`: 修正済みで、修正したバージョンが明確になっている問題です。
  * `修正したバージョン`: 問題が修正された Matlantis または関連パッケージのバージョンを記載します。未修正の場合は 未修正 とします。
  * `問題を確認しているバージョン`: 問題の発生を確認しているバージョン範囲を記載します。
