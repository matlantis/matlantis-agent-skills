# v8.0.0 ​

r2SCAN汎関数に基づく新しい計算モードR2SCANおよびR2SCAN_PLUS_D3をサポートしたモデルです。

  * r2SCAN 計算モードでは、 PBE 計算モードと比べて、形成エネルギーや融点などの固体に関する物性、反応エネルギーなどの有機分子に関する物性に関して実験値に対する再現性の向上が見込まれます。
  * PBE 計算モードのエネルギーと力の再現性がさらに向上し、相図の再現性や密度などの液体に関する物性の再現性向上が見込まれます。
  * 高密度構造におけるポテンシャルエネルギー曲面の安定性が大きく向上しました。超高圧下でのシミュレーションや結晶構造探索などにおける安定性向上が見込まれます。



当モデルは v7.0.0 でサポートされていた全ての計算モードに引き続き対応しています。

## サポートしている元素 ​

次の周期表のうち、擬ポテンシャル名が記載されている元素はPBEモードおよびPBE_Uモードで再現が可能と期待されます。このうち`*`が付いていない元素はR2SCANモードでも対応していますが、`*`付きの元素はR2SCANモードでは再現できません。R2SCANモードでは、H から Bi までのうち Pr から Lu までのランタノイドを除く70元素をサポートしています。WB97XDモードでは H, C, N, O, F, P, S, Cl, Brの9元素のみをサポートしています。

  * セルの記載は採用しているVASPのPAW擬ポテンシャル名を表します。例えば`Li_sv`は`Li_sv`ポテンシャルを、接尾辞(`_sv`、`_pv`、`_d`、`_3`)のない`Co`は標準の`Co`ポテンシャルを使用していることを表します。
  * 無印の元素: R2SCANモードおよびPBE(PBE_U)モードの両方で対応しています。
  * `*`付きの元素: PBE(PBE_U)モードのみ対応しています。R2SCANモードでは再現できません。
  * `-`と記載されている元素: いずれのモードでも対応していません。
  * 空欄は該当する元素が存在しない位置です。
