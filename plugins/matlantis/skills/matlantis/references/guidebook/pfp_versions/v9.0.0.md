# v9.0.0 ​

v8.0.0に比べて、より拡張されたr2SCAN汎関数に基づくデータセットで学習したモデルです。

  * v9.0.0 では、R2SCAN 計算モードでサポートされる元素数が PBE 計算モードと同じ 96 元素に拡張されました。
  * v9.0.0 の calc_mode のデフォルト値は R2SCAN です。



当モデルは v8.0.0 でサポートされていた全ての計算モードに引き続き対応しています。
