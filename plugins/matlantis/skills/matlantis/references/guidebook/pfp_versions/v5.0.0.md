# v5.0.0 ​

v4.0.0に比べて、より拡張されたデータセットで学習したモデルです。

  * 部分的に元素を置換した系の再現性が向上することが見込まれます
  * 分子間相互作用の再現性がさらに向上し、分子結晶や液体に関する物性の再現性が向上することが見込まれます。



当モデルはv4.0.0でサポートされていた全ての計算モードに引き続き対応しています。

## サポートしている元素 ​

次の周期表のうち、青で色付けされた元素はこのPFP versionで再現が可能と期待されます。オレンジで色付けされた元素はExperimentalなサポートであり、部分的に再現できる可能性があります。WB97XDモードでは H, C, N, O, F, P, S, Cl, Brの9元素のみをサポートしています。
