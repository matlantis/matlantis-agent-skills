# v7.0.0 ​

96元素に対応したデータセットを、改良したアーキテクチャで学習したモデルです。

  * H から U,Np,Pu,Am,Cm までの96元素に対応可能となります。



当モデルはv6.0.0でサポートされていた全ての計算モードに引き続き対応しています。

## サポートしている元素 ​

次の周期表のうち、青で色付けされた元素はこのPFP versionで再現が可能と期待されます。WB97XDモードでは H, C, N, O, F, P, S, Cl, Brの9元素のみをサポートしています。
