# v7.0.0 ​

96元素に対応したデータセットを、改良したアーキテクチャで学習したモデルです。

  * H から U,Np,Pu,Am,Cm までの96元素に対応可能となります。



当モデルはv6.0.0でサポートされていた全ての計算モードに引き続き対応しています。

## サポートしている元素 ​

次の周期表のうち、擬ポテンシャル名が記載されている元素(H〜Cmの96元素)はこのPFP versionで再現が可能と期待されます。WB97XDモードでは H, C, N, O, F, P, S, Cl, Brの9元素のみをサポートしています。

表の見方は次の通りです。

  * セルの記載は採用しているVASPのPAW擬ポテンシャル名を表します。例えば`Li_sv`は`Li_sv`ポテンシャルを、接尾辞(`_sv`、`_pv`、`_d`、`_3`)のない`Co`は標準の`Co`ポテンシャルを使用していることを表します。
  * `-`と記載されている元素は対応していません。
  * 空欄は該当する元素が存在しない位置です。
