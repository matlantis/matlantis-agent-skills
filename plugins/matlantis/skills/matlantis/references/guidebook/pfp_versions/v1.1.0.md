# v1.1.0 ​

DFT-D3補正をサポートしたモデルです。PBE_U_PLUS_D3 以外の計算モードではv1.0.0と同じふるまいになります。

## サポートしている元素 ​

データセットの対応状況の観点から元素の対応状況を以下の3段階に分割しています。基本的にはExpectedの範囲でのみ使用することを想定していますが、それ以外の元素の入力を禁止してはいません。

  * Expected: 現時点でPFPが再現する対象として主に注力している元素。このPFP versionで再現が可能と期待されます。
    * HからAsまでの33元素全て及び、Sr, Y, Zr, Nb, Mo, Ru, Rh, Pd, Ag, In, Sn, Ba, La, Ce, Gd, Hf, Ta, W, Ir, Pt, Au, Pbの合計55元素(下の周期表にて、擬ポテンシャル名が無印で記載されている元素)。
  * Experimental: 主なターゲットとしていないが、部分的に再現できる可能性があるもの。
    * HからBiまでの全ての元素のうち、上記の55元素およびランタノイド元素を除いた16元素(下の周期表にて、`*`付きで記載されている元素)。
  * Unexpected: 再現できないもの。
    * ランタノイドのうちLa, Ce, Gdを除く12元素およびPo以降の元素(下の周期表にて、括弧書き`( )`または`-`で記載されている元素)。

なお、WB97XDモードでは H, C, N, O, F, P, S, Cl, Brの9元素のみをサポートしています。

表の見方は次の通りです。

  * セルの記載は採用しているVASPのPAW擬ポテンシャル名を表します。例えば`Li_sv`は`Li_sv`ポテンシャルを、接尾辞(`_sv`、`_pv`、`_d`、`_3`)のない`Co`は標準の`Co`ポテンシャルを使用していることを表します。
  * 無印の元素: Expected(対応元素)です。
  * `*`付きの元素: Experimentalです。部分的に再現できる可能性があります。
  * 括弧書き`( )`または`-`の元素: Unexpected(再現できない元素)です。括弧書きの元素は参考として元素名・擬ポテンシャル名のみ記載しています。
  * 空欄は該当する元素が存在しない位置です。
