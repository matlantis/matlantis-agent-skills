# v4.0.0 ​

v3.0.0に比べて、より拡張されたデータセットで学習したモデルです。

  * 分子間相互作用の再現性がさらに向上し、密度などの液体に関する物性の再現性の向上が見込まれます。
  * バルク結晶構造のエネルギー順序及び密度の再現性が向上し、0Kにおける組成を変えた際の相図の再現性向上が見込まれます。



当モデルは v3.0.0 でサポートされていた全ての計算モードに引き続き対応しています。

## サポートしている元素 ​

次の周期表のうち、擬ポテンシャル名が無印で記載されている元素はこのPFP versionで再現が可能と期待されます。`*`付きの元素(Tc、Tl)はExperimentalなサポートであり、部分的に再現できる可能性があります。WB97XDモードでは H, C, N, O, F, P, S, Cl, Brの9元素のみをサポートしています。

表の見方は次の通りです。

  * セルの記載は採用しているVASPのPAW擬ポテンシャル名を表します。例えば`Li_sv`は`Li_sv`ポテンシャルを、接尾辞(`_sv`、`_pv`、`_d`、`_3`)のない`Co`は標準の`Co`ポテンシャルを使用していることを表します。
  * 無印の元素: このPFP versionで対応しています。
  * `*`付きの元素: Experimentalなサポートです。部分的に再現できる可能性があります。
  * 括弧書き`( )`の元素: このPFP versionでは対応していません(参考として元素名・擬ポテンシャル名のみ記載)。
  * `-`と記載されている元素: 対応していません。
  * 空欄は該当する元素が存在しない位置です。
